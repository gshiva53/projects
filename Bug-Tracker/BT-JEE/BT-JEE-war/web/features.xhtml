Features

Bug Tracker has the following features:

  • Application can be distributed over systems, so can be scaled both X and Y if scaled on Y

  • Reusable components

  • Responsive front-end

  • Authentication using JDBC realm

  • password protection and authentication using SHA-256

  • Authorization for different user roles

  • information communication can be achieved over the network

  • State stored in DB but stateless beans

  • Front-end uses external but powerful UI library like primefaces
